# Ferrofluid EM Display System — Concept README

## Overview
This project is a ferrofluid-based electromagnetic visual system designed to produce continuously evolving, lava-lamp-like behaviour using controlled magnetic fields generated by coil arrays beneath and/or around a shallow glass vessel (e.g. Petri dish or similar container).

The system is driven by FPGA-controlled signal generation and uses Mid/Side-style decomposition of control signals to create structured chaos rather than random motion.

---

## Physical Architecture

### Display Vessel
- Preferred: glass Petri dish (shallow, circular geometry)
- Alternative: borosilicate beaker or flat sandwich cell
- Filled with ferrofluid in low-depth layer

### Coil System
Two primary field domains:

#### 1. Radial / Bottom Coil Array
- Primary structure generator
- Defines spatial attractors
- Creates ring modes, central spikes, rotational wavefronts

#### 2. Vertical Coil Component
- Controls global compression/expansion
- Influences spike formation and collapse dynamics
- Acts as “depth pressure” on ferrofluid

Optional:
- Side coils for asymmetry and drift injection

---

## Lighting System

### LED Backlight
- LED ring (white or RGB optional)
- Positioned below diffuser layer
- Provides uniform optical illumination

### Stack Order (recommended)
1. Base plate (steel or aluminium)
2. LED ring
3. Diffusion layer (acrylic/frosted glass)
4. Coil system
5. Glass Petri dish + ferrofluid

Lighting is static (non-dynamic) to preserve visual clarity.

---

## Mechanical / Structural Base

- Steel base plate used as:
  - mechanical mounting structure
  - magnetic flux return path
  - EMI stabilisation plane

### Grounding strategy:
- Steel plate should be chassis/earth grounded at a single point
- Avoid multiple ground connections to prevent loops
- Powder coating must be locally removed for electrical contact

## Signal Processing Architecture

### Core Concept: Mono + Mid/Side Decomposition

The system operates using three conceptual signal domains:

#### Mono Core (Mid Equivalent)
- Defines stable attractor field
- Controls bulk ferrofluid structure
- Low-frequency, smooth, coherent behaviour

#### Side Component
- Introduces asymmetry and instability
- Generates motion, drift, and fragmentation
- Acts as controlled perturbation layer

#### Mid/Side Ratio Control
- Global parameter controlling balance of:
  - stability (Mid dominance)
  - chaos (Side dominance)

---

## Temporal Behaviour Design

The system is designed for **state transitions rather than fixed patterns**:

### Behaviour States
- Stable core formation (Mid dominant)
- Drifting structures (balanced Mid/Side)
- Fragmentation / multi-attractor behaviour (Side dominant)
- Recombination cycles

---

## Filter Architecture

Independent filtering applied to Mid and Side channels:

### Mid Filtering
- Controls bulk motion smoothness
- Low-pass → lava-like behaviour
- Higher bandwidth → reactive central spikes

### Side Filtering
- Controls texture and instability
- Low-pass → gentle drift distortion
- High-pass → fine fragmentation and surface agitation

### Key Principle
Filtering defines *temporal behaviour of structure*, not just frequency content.
